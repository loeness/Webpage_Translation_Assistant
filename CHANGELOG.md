# Changelog

本项目的所有重要变更都会记录在此文件中。

格式参考 Keep a Changelog，并遵循语义化版本（SemVer）。

## [1.1.0] - 2026-03-27

### Added

- 支持对翻译后页面进行原文预处理和映射。
- 支持点击单句展示对应原文。
- 支持划词后聚合展示原文片段。
- 新增生产级目录结构（assets/src/_locales）。
- 新增多语言扩展元信息（en 与 zh_CN）。
- 新增扩展图标资源（16/48/128）。
- 新增 README、LICENSE、发布模板与 Git 忽略规则。

### Changed

- 将 popup 样式从 HTML 内联拆分到独立文件。
- 调整 manifest 路径引用以匹配新目录布局。

### Fixed

- 修复内容脚本未注入时的消息连接失败问题（自动注入兜底）。
- 修复 Tooltip 原文可能被浏览器翻译器二次翻译的问题。

## [1.3.0] - 2026-03-29

### Added
- **底层架构重构**：从单节点扫描升级为“块级逻辑聚合”模型，彻底解决加粗、代码块、链接等行内标签导致的语义断裂问题。
- **虚拟长字符串投影 (Virtual Projection)**：引入实时块文本投影技术，支持 `<strong>`、`<code>`、`<a>` 标签混排下的完整原文映射。
- **绝对偏移映射系统**：通过 `TreeWalker` 实时计算文本节点在块内的绝对坐标，大幅提升点击判定精度。
- **非强制快照冻结**：增加快照保护机制，防止浏览器翻译后的 DOM 变动污染原始快照数据。

### Fixed
- 修复了在 GitHub、CSDN 等代码高亮页面中，由于 `<code>` 标签导致的原文显示不全或断句错误。
- 优化了置信度回退逻辑，当映射偏移异常时，自动回退至更高层级的块级原文，保证显示的稳定性。
